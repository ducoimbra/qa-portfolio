Projeto focado na validação de regras de negócio, contratos e tratamento de erros de uma API REST de gerenciamento de usuários.

-- Objetivo
Garantir a qualidade e a segurança dos fluxos de cadastro e autenticação, validando desde o formato das respostas até a persistência correta dos dados.

-- Ferramentas & Tecnologias
* **Postman** (Execução)
* **JSON/HTTP** (Formato e Protocolo)
* **GitHub** (Versionamento)

-- Escopo de Testes
- [x] Cadastro de usuário (Sucesso e Exceções)
- [x] Autenticação (Login e Bearer Token)
- [x] Validação de campos obrigatórios e duplicados
- [x] Verificação de Status Codes e Mensagens de Erro
- [x] Validação de cenários negativos.

--  Organização do Projeto
* `/test-cases`: Documentação detalhada dos cenários.
* `/bug-reports`: Relatórios de falhas encontradas.
* `/postman`: Collection e Environment para importação.

---
**API Utilizada:** [ServeRest](https://serverest.dev)
